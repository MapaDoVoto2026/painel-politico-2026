Sobre o Projeto

Objetivo

O Mapa do Voto 2026 é uma plataforma de transparência política que disponibiliza dados abertos sobre os parlamentares brasileiros, facilitando a análise de seu desempenho, posicionamento e integridade.

Fontes de Dados
  • Dados Abertos da Câmara
  • API do Senado Federal
  • Portal da Transparência
  • TSE - Tribunal Superior Eleitoral

Projeto open-source para transparência política | Desenvolvido para o Brasil
